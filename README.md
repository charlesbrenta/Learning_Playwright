# Learning Playwright

Small repository for Playwright learning exercises and examples.

Structure:
- Chap_01_Basics: simple JS samples used while learning Playwright setup and basics

Quick start:

1. Install Node.js (LTS) and npm.
2. From the repository root, install dependencies (if you add any):

```bash
npm install
```

3. If you use Playwright, install it and run the tests:

```bash
npx playwright install
npx playwright test
```
